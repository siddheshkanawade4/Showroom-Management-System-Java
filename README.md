# Showroom Management System

A Core Java console-based project built using OOP concepts.

## Concepts Used
- Interface
- Inheritance
- Encapsulation
- Polymorphism

## Features
- Manage showroom details
- Manage cars inventory
- Manage employee records
- No database used (in-memory data)

## Technology
- Java (Core Java)
